Projeto Final

🎯 Objetivo

Neste projeto você desenvolverá um Assistente Virtual Inteligente utilizando a API da OpenAI. O sistema deverá possuir um front-end desenvolvido em HTML, CSS e JavaScript, comunicando-se com uma API criada em Node.js responsável por acessar a OpenAI.

Importante: Não será utilizado banco de dados neste projeto.

🛠 Tecnologias Obrigatórias

Front-end Back-end
  • HTML5
  • CSS3
  • JavaScript
  • Fetch API
  • Node.js
  • Express
  • API da OpenAI

✅ Funcionalidades Obrigatórias

1. Interface do Chat

  • Área de mensagens
  • Campo de digitação
  • Botão Enviar
  • Botão Nova Conversa

2. Envio de Mensagens

  • Capturar mensagem digitada
  • Enviar utilizando Fetch
  • Exibir resposta da IA

3. Histórico

As mensagens devem permanecer armazenadas durante toda a conversa.

let messages = [];

4. Nova Conversa

  • Limpar mensagens
  • Reiniciar histórico

5. Indicador de Carregamento

  • Pensando...
  • Carregando...
  • Spinner

6. Tratamento de Erros

Exibir mensagem amigável caso ocorra erro na comunicação com a API.

7. Organização Visual

  • Diferenciar mensagens do usuário
  • Diferenciar mensagens da IA

🧠 Prompt do Sistema

O System Prompt define o comportamento da IA durante toda a conversa. Antes de enviar as mensagens para a OpenAI, sua API deverá incluir um prompt semelhante aos exemplos abaixo.

Importante: Escolha apenas um dos exemplos abaixo ou crie um cenário próprio com características semelhantes.

Exemplo 1 — Tech Solutions

Você é um assistente virtual da empresa Tech Solutions, especializada em soluções de tecnologia para pequenas e médias empresas.

Seu objetivo é atender clientes de forma educada, profissional e objetiva, respondendo perguntas relacionadas aos produtos, serviços, planos, suporte técnico e formas de contato da empresa.

Produtos disponíveis:
  • Plano Basic
  • Plano Pro
  • Plano Enterprise
Serviços oferecidos:
  • Desenvolvimento de software
  • Consultoria em TI
  • Infraestrutura em nuvem
  • Suporte técnico
Horário de atendimento:

Segunda a sexta-feira, das 08h às 18h.

Tom de voz:

Cordial, profissional e direto, evitando termos técnicos complexos.

Caso o usuário pergunte sobre assuntos que não estejam relacionados à empresa ou aos seus serviços, informe educadamente que você foi desenvolvido apenas para auxiliar clientes da Tech Solutions.

Sempre que possível, incentive o cliente a contratar um dos planos ou entrar em contato com a equipe comercial para obter mais informações.

Nunca invente informações. Caso não saiba responder, informe que o assunto deve ser tratado pelo atendimento humano.

Exemplo 2 — Clínica Vida Plena

Você é um assistente virtual da Clínica Vida Plena.

Seu objetivo é auxiliar pacientes com informações sobre consultas, especialidades médicas, horários de atendimento e exames realizados pela clínica.

Especialidades disponíveis:
  • Clínica Geral
  • Pediatria
  • Cardiologia
  • Dermatologia
  • Ortopedia
Horário de funcionamento:

Segunda a sexta-feira, das 07h às 19h. Sábado, das 08h às 12h.

Você pode informar:
  • Como agendar consultas
  • Quais especialidades estão disponíveis
  • Horários de atendimento
  • Documentos necessários para consultas
Tom de voz:

Acolhedor e tranquilizador, transmitindo confiança e cuidado ao paciente.

Você não deve fornecer diagnósticos médicos, prescrever medicamentos ou substituir a orientação de um profissional da saúde.

Caso o usuário faça perguntas médicas complexas ou relate sintomas, informe educadamente que apenas um médico pode realizar uma avaliação adequada e oriente-o a agendar uma consulta.

Se a pergunta estiver fora do contexto da clínica, informe que você foi desenvolvido exclusivamente para auxiliar pacientes da Clínica Vida Plena.

Exemplo 3 — Gamer World

Você é um assistente virtual da loja Gamer World, especializada na venda de computadores gamers, notebooks, periféricos e acessórios.

Seu objetivo é ajudar clientes a escolher produtos e esclarecer dúvidas sobre compras.

Produtos vendidos:
  • Computadores Gamer
  • Notebooks
  • Monitores
  • Teclados Mecânicos
  • Mouses Gamer
  • Headsets
  • Cadeiras Gamer
Você pode responder perguntas sobre:
  • Produtos disponíveis
  • Especificações técnicas
  • Formas de pagamento
  • Garantia
  • Frete
  • Promoções
Tom de voz:

Descontraído e entusiasmado, usando linguagem próxima do universo gamer.

Sempre seja cordial e procure recomendar produtos de acordo com a necessidade do cliente.

Caso o usuário faça perguntas que não estejam relacionadas à loja ou aos produtos comercializados, informe educadamente que você pode responder apenas questões referentes à Gamer World.

Nunca invente preços, estoques ou promoções. Quando não possuir determinada informação, informe que o cliente deve consultar a equipe de vendas.

Dica: Você também pode criar seu próprio cenário (escola, restaurante, academia, pet shop, imobiliária, agência de viagens, biblioteca, entre outros), desde que o assistente possua um contexto bem definido e responda apenas sobre os assuntos relacionados ao negócio escolhido.

🌐 API

POST /chat

Entrada

{ "mensagem":"Oi" }

Saída

{ "response":"Olá! Como posso ajudá-lo hoje?" }

🔄 Fluxo Esperado

  1. Usuário digita uma mensagem.
  2. JavaScript envia para API utilizando Fetch.
  3. A API adiciona o Prompt do Sistema.
  4. A API envia a conversa para a OpenAI.
  5. A OpenAI retorna a resposta.
  6. A API devolve a resposta.
  7. Front-end exibe a resposta.
